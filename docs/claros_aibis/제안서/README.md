## 제안서 구조

### 공통 섹션

1. **Executive Summary** - 요약
2. **회사/팀 소개** - 비전, 핵심 역량
3. **시장 분석** - 시장 규모, 경쟁사, SWOT
4. **솔루션 개요** - claros_aibis 소개, 차별성
5. **비즈니스 모델** - 수익 모델, 가격 정책
6. **ROI/효과 분석** - 투자 대비 효과
7. **도입 방법론** - 구축 계획, 일정

### 4가지 포지셔닝

1. **스마트팩토지 전문 BI** - 생산, 품질, 설비 KPI 통합 관리
2. **중소기업용 가성비 ERP+BI** - SaaS 구독형, 저진입
3. **AI 예측 분석 전문** - AI/ML 기반 예측, 이상 탐지
4. **ERP 연동 통합 플랫폼** - SAP/FOM 등 기존 ERP 연동
