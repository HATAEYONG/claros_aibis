# claros_aibis 마스터 제안서

## Executive Summary

### 비즈니스 기회

제조 기업의 디지털 전환 가속화와 스마트팩토리 도입 확대로, 중소 제조업체를 중심으로 **비용 효율적이고 빠른 도입이 가능한 KPI 관리 솔루션**의 시장이 급성장하고 있습니다. 기존 ERP/BI 솔루션은 대기업 중심이고 고가/복잡하여 중소기업은 진입 장벽이 높습니다.

### 솔루션 가치 제안

**claros_aibis**는 이러한 시장 기회를 포착하여, **3가지 핵심 차별성**을 제공합니다:

| 차별점 | 설명 |
|--------|------|
| 저진입 | 월 30만원부터 시작하는 SaaS 구독형, 2주 내 빠른 구축 |
| 쉬운 사용 | ERP 데이터 자동 연동, 클릭 몇 번으로 KPI 대시보드 생성 |
| AI 기반 | 예측/이상 탐지로 사전 예방, 문제 발생 전 대응 |

### 주요 혜택

| 혜택 | 설명 |
|------|------|
| 비용 절감 | 불량률 15% 감소, 재고비 20% 절감, 가동률 10% 향상 |
| 생산성 향상 | 실시간 KPI 가시화, 의사결정 시간 50% 단축 |
| 품질 개선 | AI 예측 분석으로 불량 원인 사전 탐지 |
| 스마트팩토리 | 정부 지원사업 연계, 컨설팅 일관 제공 |

## 3. 솔루션 개요

### 3.1 claros_aibis 소개

**claros_aibis**는 제조 기업의 **KPI(핵심성과지표) 통합 관리**를 위한 클라우드 기반 BI 솔루션입니다.

**핵심 가치**:
1. **ERP 데이터 자동 연동** - SAP/FOM 등 기존 ERP 데이터 자동 수집
2. **80개 KPI 레지스트리** - 재무/원가/생산/품질/영업/구매/제조/관리회계
3. **AI 기반 예측/이상 탐지** - 문제를 사전에 예측하고 대응
4. **가시화 대시보드** - 경영진, 현장 관리자용 맞춤형 대시보드

### 3.2 핵심 기능

| 기능 | 설명 |
|------|------|
| 마스터 데이터 관리 | 계정과목, 창고, 공정, 은행 등 기준정보 CRUD |
| KPI 관리 | 8개 카테고리 80개 KPI 정의, 실적 조회, 재계산 |
| ERP 데이터 통합 | SAP/FOM 연동, 실시간/배치 데이터 수집 |
| AI 예측 분석 | 매출/생산/품질 예측, 인과 분석, 이상 탐지 |
| 대시보드 | 경영 대시보드, 컨트롤 타워, KPI 카드 |

### 3.4 4가지 포지셔닝

#### 포지셔닝 1: 스마트팩토지 전문 BI

**타겟**: 스마트팩토리 도입을 고려하는 중소 제조업체
**핵심 메시지**: "생산/품질/설비 KPI를 한눈에 관리하세요"

**제공 가치**:
- 8개 카테고리 80개 KPI 표준 제공
- 실시간 KPI 대시보드
- 컨트롤 타워로 경영진 통합 관제

---

#### 포지셔닝 2: 중소기업용 가성비 ERP+BI

**타겟**: ERP 보유하나 BI 부족한 중소 제조업체
**핵심 메시지**: "보유 ERP 그대로 활용, 월 30만원부터"

**제공 가치**:
- SAP/FOM 연동으로 추가 ERP 구축 불필요
- SaaS 구독형으로 초기 투자 최소화
- 2주 내 빠른 구축

---

#### 포지셔닝 3: AI 예측 분석 전문

**타겟**: AI/ML 도입을 원하나 진입 장벽이 높은 기업
**핵심 메시지**: "AI 전문가 없어도 예측/이상 탐지 자동화"

**제공 가치**:
- 매출/생산/품질 예측
- 이상 탐지로 사전 예방
- 인과 분석으로 원인 규명

---

#### 포지셔닝 4: ERP 연동 통합 플랫폼

**타겟**: 다중 ERP 보유 기업 (SAP+FOM 등)
**핵심 메시지**: "여러 ERP를 하나로 통합, KPI 통합 관리"

**제공 가치**:
- SAP/FOM 데이터 통합
- 마스터 데이터 표준화
- KPI 통합 계산
