## 3. ViewSet 구현

### 3.1 마스터 데이터 ViewSet

**파일**: `erp_sync/data_hub/master/views.py`

**상속 구조**:
```python
class MasterAccountViewSet(ExportImportMixin, BatchOperationsMixin, viewsets.ModelViewSet):
    """
    ExportImportMixin:
        - export_csv()
        - export_excel()
        - export_json()
        - import_data()

    BatchOperationsMixin:
        - batch_create()
        - batch_update()
        - batch_delete()

    ModelViewSet:
        - list()
        - retrieve()
        - create()
        - update()
        - partial_update()
        - destroy()
    """
```

**구현된 ViewSet (10개)**:
- MasterProductViewSet
- MasterVendorViewSet
- MasterCustomerViewSet
- MasterDepartmentViewSet
- MasterEmployeeViewSet
- MasterEquipmentViewSet
- MasterAccountViewSet (신규)
- MasterWarehouseViewSet (신규)
- MasterProcessViewSet (신규)
- MasterBankViewSet (신규)

**공통 기능**:
- 필터: DjangoFilterBackend
- 검색: SearchFilter (code, name)
- 정렬: OrderingFilter
- 내보내기: CSV, Excel, JSON
- 가져오기: CSV, Excel, JSON
- 일괄 작업: create, update, delete

```python
# config/urls.py
urlpatterns = [
    path('admin/', admin.site.urls),
    path('api/', include('data_hub.urls')),
    path('swagger/', schema_view.with_ui('swagger', cache_timeout=0)),
    path('redoc/', schema_view.with_ui('redoc', cache_timeout=0)),

# data_hub/urls.py
urlpatterns = [
    path('data-hub/master/', include('erp_sync.data_hub.master.urls')),

# master/urls.py
router = DefaultRouter()
router.register(r'accounts', MasterAccountViewSet, basename='account')
```
