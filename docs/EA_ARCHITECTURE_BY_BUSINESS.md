## 2. 비즈니스 아키텍처 (업무 구조)

### 2.1 업무 도메인 구성

#### **도메인 1: 제조 운영 (Manufacturing Operations)**

| 하위 도메인 | 핵심 프로세스 | 주요 KPI | 관련 시스템 |
|------------|--------------|----------|-----------|
| 생산 관리 | 작업 지시, 생산 실적, BOM 관리 | 생산량, 가동률, OEE | Production Module |
| 제조 관리 | 공정 관리, 작업 표준, 설비 관리 | 사이클 타임, 가동률 | Manufacturing Module |
| 설비 관리 | 설비 보전, 예방 정비, 고장 관리 | MTBF, MTTR, 가동률 | Equipment Module |

#### **도메인 2: 품질 관리 (Quality Management)**

| 하위 도메인 | 핵심 프로세스 | 주요 KPI | 관련 시스템 |
|------------|--------------|----------|-----------|
| 품질 검사 | 검사 관리, 불량 관리, 검사 기준 | 일차 수율, 고객 불만 | Quality Module |
| 품질 원가 | 예방 비용, 평가 비용, 실패 비용 | 품질 비용율, 불량률 | Quality Cost Module |
| 원인 분석 | 6M 시나리오 분석, 불량 원인 추적 | 해결 시간, 재발 방지 | 6M Scenario Module |

#### **도메인 3: 공급망 운영 (Supply Chain Operations)**

| 하위 도메인 | 핵심 프로세스 | 주요 KPI | 관련 시스템 |
|------------|--------------|----------|-----------|
| 구매 관리 | 발주, 협력사 관리, 구매 원가 | 납기 준수율, 재고 회전율 | Purchase Module |
| 자재 관리 | 자재 수급, 재고 관리 | 재고 회전율, 안전 재고 | Material Module |
| 물류 관리 | 배송, 창고 관리 | 배송 정시율, 창고 효율 | Logistics Module |
| P2P 프로세스 | 구매 요청 ~ 지급까지 | 리드 타임, 구매 비용 | P2P Module |

#### **도메인 4: 재무/회계 (Finance & Accounting)**

| 하위 도메인 | 핵심 프로세스 | 주요 KPI | 관련 시스템 |
|------------|--------------|----------|-----------|
| 재무 관리 | 재무 제표, 손익 분석 | 매출액, 영업이익률 | Financial Module |
| 원가 관리 | 표준 원가, 원가 분석 | 원가 차이, 원가 절감액 | Cost Module |
| 관리 회계 | 예산 관리, 성과 평가 | 예산 차이, ROI | Managerial Accounting Module |

#### **도메인 5: 영업/고객 (Sales & Customer)**

| 하위 도메인 | 핵심 프로세스 | 주요 KPI | 관련 시스템 |
|------------|--------------|----------|-----------|
| 영업 관리 | 매출, 수주, 고객 관리 | 매출 증가율, 신규 고객 | Sales Module |
| 견적 원가 | 제품별 원가 분석 | 마진율, 수익성 | Sales Cost Module |
| O2C 프로세스 | 주문 접수 ~ 입금까지 | 주문 처리 시간, 매출 | O2C Module |

#### **도메인 6: 개발/혁신 (R&D & Innovation)**

| 하위 도메인 | 핵심 프로세스 | 주요 KPI | 관련 시스템 |
|------------|--------------|----------|-----------|
| 개발 관리 | R&D 프로젝트, 설계 관리 | 프로젝트 달성률, 신제품 수 | Development Module |
| 설계 원가 | 설계 비용, 자재비 분석 | 설계 원가 절감액 | Design Cost Module |
| 외주 원가 | 협력사 외주비 분석 | 외주비, 품질 등급 | Outsourcing Cost Module |

#### **도메인 7: 인적자원 (Human Resources)**

| 하위 도메인 | 핵심 프로세스 | 주요 KPI | 관련 시스템 |
|------------|--------------|----------|-----------|
| 인사 관리 | 직원, 부서 관리 | 인원 현황, 이동률 | HR Module |
| 급여 관리 | 급여, 수당 관리 | 급여 지급 정확도 | Payroll Module |

#### **도메인 8: ESG 경영 (Sustainability)**

| 하위 도메인 | 핵심 프로세스 | 주요 KPI | 관련 시스템 |
|------------|--------------|----------|-----------|
| 환경 경영 | 탄소 배출, 에너지 절감, 재활용 | 탄소 배출량, 에너지 절감율 | ESG Module |
| 사회 책임 | 직원 만족도, 안전 보건 | 직원 만족도, 안전 사고 | ESG Module |
| 지배 구조 | 윤리 경영, 투명성 | 지배 구조 점수 | ESG Module |

## 4. 애플리케이션 아키텍처 (도메인 서비스)

### 4.1 도메인 서비스 구성

#### **도메인 서비스 1: 제조 서비스 (Manufacturing Service)**

```
ProductionModule (생산 관리)
├── api/production/
│   ├── work-orders/          # 작업 지시
│   ├── production-results/   # 생산 실적
│   ├── bom-management/       # BOM 관리
│   └── kpi/all_kpis/         # 생산 KPI
├── models/
│   ├── WorkOrder (작업 지시서)
│   ├── ProductionResult (생산 실적)
│   └── BOM (재료 명세서)
└── services/
    ├── ProductionService (생산 서비스)
    └── WorkOrderService (작업 지시 서비스)
```

#### **도메인 서비스 2: 품질 서비스 (Quality Service)**

```
QualityModule (품질 관리)
├── api/quality/
│   ├── inspections/           # 검사 관리
│   ├── defects/               # 불량 관리
│   ├── quality-reports/       # 품질 보고서
│   └── kpi/all_kpis/          # 품질 KPI
├── models/
│   ├── Inspection (검사)
│   ├── Defect (불량)
│   └── QualityReport (품질 보고서)
└── services/
    ├── InspectionService (검사 서비스)
    └── DefectAnalysisService (불량 분석 서비스)
```

#### **도메인 서비스 3: 원가 서비스 (Cost Service)**

```
CostModule (원가 관리)
├── api/cost/
│   ├── standard-costs/        # 표준 원가
│   ├── cost-analysis/         # 원가 분석
│   ├── variance-analysis/      # 차이 분석
│   └── abc-costing/           # ABC 원가
├── models/
│   ├── StandardCost (표준 원가)
│   ├── CostVariance (원가 차이)
│   └── ABCCost (ABC 원가)
└── services/
    ├── StandardCostService (표준 원가 서비스)
    └── CostAnalysisService (원가 분석 서비스)
```

## 5. 기술 아키텍처

### 5.1 시스템 아키텍처 레이어

```
┌─────────────────────────────────────────────────────────────────┐
│  PRESENTATION LAYER (프레젠테이션 레이어)                      │
│  ┌────────────────────────────────────────────────────────────┐ │
│  │  React 18 + TypeScript + TailwindCSS + Chart.js           │ │
│  │  - 16 업무 도메인 컴포넌트                                  │ │
│  │  - AI 챗봇, 대시보드, KPI 관리                              │ │
│  └────────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────┘
                            │ HTTP/REST
                            ▼
┌─────────────────────────────────────────────────────────────────┐
│  API GATEWAY LAYER (API 게이트웨이 레이어)                      │
│  ┌────────────────────────────────────────────────────────────┐ │
│  │  Django REST Framework + JWT + CORS                        │ │
│  │  - 인증/인가 (JWT)                                         │ │
│  │  - API 라우팅 (도메인별)                                  │ │
│  │  - 요청/응답 로깅                                          │ │
│  └────────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────┘
                            │
                            ▼
┌─────────────────────────────────────────────────────────────────┐
│  BUSINESS LOGIC LAYER (비즈니스 로직 레이어)                    │
│  ┌────────────────────────────────────────────────────────────┐ │
│  │  16 도메인 서비스 + ERP 연동 서비스                          │ │
│  │  - ProductionService, QualityService, FinancialService,   │ │
│  │  - CostService, SalesService, PurchaseService, etc.       │ │
│  │  - ERP Sync Service (YH, FOM, SAP 연동)                   │ │
│  └────────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────┘
                            │
                            ▼
┌─────────────────────────────────────────────────────────────────┐
│  DATA ACCESS LAYER (데이터 접근 레이어)                        │
│  ┌────────────────────────────────────────────────────────────┐ │
│  │  Django ORM + QuerySet + Aggregation API                   │ │
│  │  - 데이터 모델 정의 (ORM)                                  │ │
│  │  - 쿼리 최적화                                             │ │
│  │  - 집계/분석 API                                          │ │
│  └────────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────┘
                            │
                            ▼
┌─────────────────────────────────────────────────────────────────┐
│  DATA STORAGE LAYER (데이터 저장 레이어)                        │
│  ┌────────────────────────────────────────────────────────────┐ │
│  │  PostgreSQL (운영) + SQLite (개발) + Redis (캐시)        │ │
│  │  - ERP 데이터 연동 (YH:PostgreSQL, FOM:MSSQL, SAP:Oracle)│ │
│  └────────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────┘
```
